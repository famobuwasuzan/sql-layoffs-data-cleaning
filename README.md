# Global Layoffs Data Cleaning (MySQL)

## Project Overview
This project focuses on cleaning a global layoffs dataset using MySQL. 
The dataset contained duplicates, missing values, inconsistent text fields, and incorrect data types.

## Tools Used
- MySQL
- SQL (Window Functions, Joins, Data Type Conversion)

## Cleaning Steps
- Created a working copy of raw data
- Removed duplicate records using ROW_NUMBER()
- Standardised industry and country values
- Converted date column from TEXT to DATE
- Handled missing values
- Removed irrelevant rows

## Key Learning
This project strengthened my understanding of data integrity, deduplication logic, and preparing raw data for analysis.
